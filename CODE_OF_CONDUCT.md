# Code of Conduct

This project follows a simple rule set for collaboration:

- Be respectful and technical in discussions.
- Critique ideas and implementations, not people.
- Provide reproducible evidence when reporting issues.
- Do not post private data, secrets, or personal attacks.

Maintainers may moderate comments, issues, or PRs that violate these rules.

## 中文说明

本项目采用以下协作规则：

- 保持尊重，讨论聚焦技术事实。
- 可以质疑方案，不攻击个人。
- 反馈问题时尽量提供可复现依据。
- 禁止发布隐私数据、密钥或人身攻击内容。

维护者有权对违规 issue/PR/评论进行处理。
