# BRIEF Backend

Fastify backend service for `POST /api/brief`.

## Start

```bash
npm --prefix backend install
npm --prefix backend run dev
```

Default port: `8787` (configurable by `PORT`).

## Build

```bash
npm --prefix backend run build
```

## Environment

- Template file: `backend/env.example` (placeholders only)
- Local runtime values: `backend/.env` or repo root `.env`
